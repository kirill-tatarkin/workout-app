# 🏋️ Workout Tracker PWA

Трекер тренировок PPL с таймером отдыха и звуковым сигналом.

## Деплой на Vercel (бесплатно, 5 минут)

### 1. Зарегистрируйся на Vercel
Перейди на [vercel.com](https://vercel.com) → Sign Up → Continue with GitHub

### 2. Загрузи проект
Вариант А — через GitHub:
1. Создай репозиторий на [github.com](https://github.com)
2. Загрузи все файлы этой папки
3. На Vercel: New Project → Import Git Repository

Вариант Б — через Vercel CLI (проще):
```bash
npm i -g vercel
vercel
```
Следуй инструкциям, через минуту получишь ссылку типа `workout-tracker.vercel.app`

### 3. Установи на iPhone как приложение
1. Открой ссылку в **Safari** (не Chrome!)
2. Нажми кнопку «Поделиться» (квадрат со стрелкой снизу)
3. Выбери «На экран "Домой"»
4. Нажми «Добавить»

Готово — иконка появится на рабочем столе как настоящее приложение!

## Локальный запуск
```bash
npm install
npm run dev
```

## Сборка
```bash
npm run build
```
